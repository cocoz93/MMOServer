구체적 예시 — Refresh가 만료 시점을 리셋한다
timeout 60초 · tick 5초 · 슬롯 13개 · Session A의 수명 추적
0
0초 접속 · cursor=0
Session A를 Slot 0에 삽입 (한 바퀴 뒤 만료 예정)
↻
30초 패킷 수신 → Refresh
Slot 0에서 빼고 Slot 6으로 재삽입 → 만료 시점 리셋
✕
90초 cursor가 Slot 6 도달 → 만료
RequestDisconnectSession(A) 호출
마지막 활동 30초 + 타임아웃 60초 = 90초 만료
Refresh가 없었다면  0 + 60 = 60초  에 만료됐을 것
노드가 삽입된 슬롯에 cursor가 한 바퀴 돌아 다시 도달하면 만료 · Refresh는 노드를 현재 cursor 슬롯으로 옮겨 타이머를 리셋 · 한 바퀴 = 12 tick × 5초 = 60초
